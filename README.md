# 🛡️ Anti-Kuddus Protocol

A modern student complaint management system built with HTML, CSS and JavaScript.

---

## 📖 About

Anti-Kuddus Protocol is a web application that helps students submit complaints anonymously, manage seating plans, send SOS alerts, and view complaint analytics through an easy-to-use dashboard.

---

## ✨ Features

- 🔐 Login System
- 📋 Complaint Management
- 📊 Dashboard
- ⚠️ Warning System
- 🔍 Search & Filter
- ✏️ Edit & Delete Complaints
- 🪑 Smart Seat Planner
- 🆘 SOS Emergency
- 👨‍💼 Admin Dashboard
- 📈 Analytics Dashboard

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage
- Chart.js
- Font Awesome
- Google Fonts
